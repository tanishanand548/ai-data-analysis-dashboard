# AI Data Analyst Dashboard 📊

An interactive data analysis web application built using Python and Streamlit that allows users to upload datasets, analyze them, visualize data, and run SQL queries.

---

## 🚀 Live Demo
https://aidatadashboard.streamlit.app/

---

## 📌 Features

- Upload CSV files
- Automatic data preview
- Statistical summary of dataset
- Interactive line and histogram charts
- SQL query execution on dataset

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib
- SQLite

---

## 📂 Project Structure

    ai-data-dashboard/
    │
    ├── app.py
    ├── requirements.txt
    └── data/


---

## ▶️ How to Run Locally

```bash
git clone https://github.com/your-username/ai-data-dashboard.git

cd ai-data-dashboard

pip install -r requirements.txt

streamlit run app.py
```
